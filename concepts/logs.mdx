To ensure accountability and transparency, Inquira provides detailed call transcripts which include:
- Every word spoken by the AI agent (untill the call was disconnected)
- Every word understood by the AI agent
- Every function call made by the AI agent
- Every input collected by the AI agent linked to the transcript
- Analysers configured by the user to analyse the call transcript
